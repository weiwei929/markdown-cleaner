# 测试文档

这是一个用来测试引号错位修复功能的文档。

## 引号问题示例

1. 基本对话
   - 张三说："你好"，我很高兴见到你。
   - 李四回答："我也很高兴！"

2. 引号错位问题
   - 他说："这是左引号开始，但用了右引号结束"
   - 她问："为什么会出现这种情况？"

3. 连续相同引号
   - 错误格式：他说：""你好""，这样不对。
   - 正确格式应该是：他说："你好"，这样才对。

4. 嵌套引号
   - 老师说："学生问'什么是正确的引号用法？'，这是一个好问题。"

5. 引号与标点
   - 问句："你明白了吗？"
   - 感叹句："太好了！"
   - 陈述句："我明白了。"

这个文档包含了各种引号使用情况，可以用来测试修复功能。
